§7.2 Integration by Parts

Substitution came from the chain rule. From the product rule we get integration by parts.
Let u = u ( x ) u = u ( x ) u=u(x)u=u(x)u=u(x) and v = v ( x ) v = v ( x ) v=v(x)v=v(x)v=v(x) be two differentiable functions of x x xxx. By the product rule,
d d x ( u v ) = u ′ ( x ) v ( x ) + u ( x ) v ′ ( x ) d d x ( u v ) = u ′ ( x ) v ( x ) + u ( x ) v ′ ( x ) (d)/(dx)(uv)=u^(')(x)v(x)+u(x)v^(')(x)\frac{d}{d x}(u v)=u^{\prime}(x) v(x)+u(x) v^{\prime}(x)ddx(uv)=u′(x)v(x)+u(x)v′(x)

This last example illustrates two important features of integration by parts.

Important note: integrals of the type ∫ e a x sin ⁡ ( b x ) d x ∫ e a x sin ⁡ ( b x ) d x inte^(ax)sin(bx)dx\int e^{a x} \sin (b x) d x∫eaxsin⁡(bx)dx and ∫ e a x cos ⁡ ( b x ) d x ∫ e a x cos ⁡ ( b x ) d x inte^(ax)cos(bx)dx\int e^{a x} \cos (b x) d x∫eaxcos⁡(bx)dx

You need two integrations by parts, and in the first integration by parts you can let u u uuu be either the exponential function or the trigonometric function. However, the second integration by parts must be "in the same direction" as the first.
In other words, if u u uuu is the exponential function in the first integration, then the new u u uuu for the second should again be the exponential function. Similarly, if you pick the trigonometric function for u u uuu in the first integration, you need to pick the trigonometric function again for u u uuu in the second.
If you don't do this, you end up undoing the first integration and saying something like
∫ e a x sin ⁡ ( b x ) d x = ∫ e a x sin ⁡ ( b x ) d x ∫ e a x sin ⁡ ( b x ) d x = ∫ e a x sin ⁡ ( b x ) d x inte^(ax)sin(bx)dx=inte^(ax)sin(bx)dx\int e^{a x} \sin (b x) d x=\int e^{a x} \sin (b x) d x∫eaxsin⁡(bx)dx=∫eaxsin⁡(bx)dx
which is true, but useless. Try it!
After the second integration by parts, a constant multiple of the original integral appears on the right-hand side. This gives an equation in the desired integral, which can then be solved by simple algebra.
