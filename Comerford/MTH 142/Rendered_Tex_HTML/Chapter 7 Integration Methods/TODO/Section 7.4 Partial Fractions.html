Partial Fractions

Section 7.4 Algebraic Identities and Trigonometric Substitutions

Part I --- The method of partial fractions

This is a method for rewriting certain rational functions in a way which makes them
easier to integrate.

Example

Find A , B for which
1 ( x − 2 ) ( x − 5 ) = A x − 2 + B x − 5 .
Put the right-hand side over a common denominator:
A x − 2 + B x − 5 = A ( x − 5 ) ( x − 2 ) ( x − 5 ) + B ( x − 2 ) ( x − 2 ) ( x − 5 ) .
So
1 ( x − 2 ) ( x − 5 ) = A ( x − 5 ) + B ( x − 2 ) ( x − 2 ) ( x − 5 ) .
Multiply both sides by ( x − 2 ) ( x − 5 ) :
1 = A ( x − 5 ) + B ( x − 2 ) .
1 = ( A + B ) x − 5 A − 2 B .
The left-hand side and right-hand side are both polynomials in x , and they are equal
iff the coefficients of the different powers of x match. So
A + B = 0 ( 1 ) − 5 A − 2 B = 1 ( 2 )
From (1), B = − A . Substituting into (2) gives
− 5 A + 2 A = 1 ⇒ − 3 A = 1 ⇒ A = − 1 3 ,
and so
B = − A = 1 3 .
Therefore
1 ( x − 2 ) ( x − 5 ) = − 1 3 ( x − 2 ) + 1 3 ( x − 5 ) .
Now let's see why this is so useful.

Example

∫ 1 ( x − 2 ) ( x − 5 ) d x .
From above we can rewrite this as
∫ ( − 1 3 ( x − 2 ) + 1 3 ( x − 5 ) ) d x = − 1 3 ∫ d x x − 2 + 1 3 ∫ d x x − 5 .
= − 1 3 ln ⁡ | x − 2 | + 1 3 ln ⁡ | x − 5 | + C .

Example

∫ x + 2 x 2 + x d x .
Factor the denominator x 2 + x as x ( x + 1 ) and write
x + 2 x 2 + x = x + 2 x ( x + 1 ) = A x + B x + 1 .
Then
x + 2 x ( x + 1 ) = A ( x + 1 ) + B x x ( x + 1 ) .
So
x + 2 = A ( x + 1 ) + B x = ( A + B ) x + A .
Thus
A + B = 1 ( 1 ) A = 2 ( 2 )
Substituting (2) into (1) gives 2 + B = 1 , so B = − 1 . Hence
x + 2 x 2 + x = 2 x − 1 x + 1 .
and so
∫ x + 2 x 2 + x d x = 2 ∫ d x x − ∫ d x x + 1 = 2 ln ⁡ | x | − ln ⁡ | x + 1 | + C .

Example

∫ 10 x − 2 x 2 ( x − 1 ) 2 ( x + 3 ) d x .
Note. Although the ( x − 1 ) term is squared, we will need a 1 x − 1
term in our partial fraction expansion as well as a 1 ( x − 1 ) 2 term.
So write
10 x − 2 x 2 ( x − 1 ) 2 ( x + 3 ) = A x − 1 + B ( x − 1 ) 2 + C x + 3 .
Then
10 x − 2 x 2 ( x − 1 ) 2 ( x + 3 ) = A ( x − 1 ) ( x + 3 ) ( x − 1 ) 2 ( x + 3 ) + B ( x + 3 ) ( x − 1 ) 2 ( x + 3 ) + C ( x − 1 ) 2 ( x − 1 ) 2 ( x + 3 ) .
So
10 x − 2 x 2 = A ( x − 1 ) ( x + 3 ) + B ( x + 3 ) + C ( x − 1 ) 2 .
Expanding,
10 x − 2 x 2 = A ( x 2 + 2 x − 3 ) + B ( x + 3 ) + C ( x 2 − 2 x + 1 ) .
Collecting like powers of x ,
10 x − 2 x 2 = ( A + C ) x 2 + ( 2 A + B − 2 C ) x + ( − 3 A + 3 B + C ) .
Compare powers of x :
A + C = − 2 ( 1 ) 2 A + B − 2 C = 10 ( 2 ) − 3 A + 3 B + C = 0 ( 3 )
Solve by Gaussian elimination. Take 2 ⋅ ( 1 ) from (2) and add 3 ⋅ ( 1 ) to (3)
to get rid of A in (2) and (3):
A + C = − 2 ( 1 ) B − 4 C = 14 ( 2 ) ′ 3 B + 4 C = − 6 ( 3 ) ′
Take 3 ⋅ ( 2 ) ′ from ( 3 ) ′ to get rid of B in ( 3 ) ′ :
A + C = − 2 ( 1 ) B − 4 C = 14 ( 2 ) ′ 16 C = − 48 ( 3 ) ″
Now do back substitution to get C , B , A in turn.
From ( 3 ) ″ , C = − 3 . Substituting for C into ( 2 ) ′ gives
B − 4 ( − 3 ) = 14 ⇒ B = 2.
Substituting for B and C into (1) gives
A − 3 = − 2 ⇒ A = 1.
So A = 1 , B = 2 , C = − 3 , and
10 x − 2 x 2 ( x − 1 ) 2 ( x + 3 ) = 1 x − 1 + 2 ( x − 1 ) 2 − 3 x + 3 .
The actual integration is now fairly easy:
∫ 10 x − 2 x 2 ( x − 1 ) 2 ( x + 3 ) d x = ∫ ( 1 x − 1 + 2 ( x − 1 ) 2 − 3 x + 3 ) d x .
In the first two integrals let w = x − 1 (so d w = d x ), and in the third let y = x + 3
(so d y = d x ). We get
∫ d w w + 2 ∫ d w w 2 − 3 ∫ d y y = ln ⁡ | w | − 2 w − 3 ln ⁡ | y | + C .
Converting back to x and y ,
ln ⁡ | x − 1 | − 2 x − 1 − 3 ln ⁡ | x + 3 | + C .

Example

∫ 2 x 2 − x − 1 ( x 2 + 1 ) ( x − 2 ) d x .
Here x 2 + 1 cannot be factored any further (using only real numbers). The correct
partial fractions expansion to look for in this case is of the form
2 x 2 − x − 1 ( x 2 + 1 ) ( x − 2 ) = A x + B x 2 + 1 + C x − 2 .
Then
A x + B x 2 + 1 + C x − 2 = ( A x + B ) ( x − 2 ) + C ( x 2 + 1 ) ( x 2 + 1 ) ( x − 2 ) .
So
2 x 2 − x − 1 = ( A x + B ) ( x − 2 ) + C ( x 2 + 1 ) .
Expanding,
2 x 2 − x − 1 = A x 2 + ( − 2 A + B ) x − 2 B + C x 2 + C .
Thus
2 x 2 − x − 1 = ( A + C ) x 2 + ( − 2 A + B ) x + ( − 2 B + C ) .
Equating powers of x gives
A + C = 2 ( 1 ) − 2 A + B = − 1 ( 2 ) − 2 B + C = − 1 ( 3 )
Gaussian elimination (again!). Add 2 ⋅ ( 1 ) to (2) to get rid of A in (2):
A + C = 2 ( 1 ) B + 2 C = 3 ( 2 ) ′ − 2 B + C = − 1 ( 3 )
Add 2 ⋅ ( 2 ) ′ to (3) to get rid of B in (3):
A + C = 2 ( 1 ) B + 2 C = 3 ( 2 ) ′ 5 C = 5 ( 3 ) ″
Back substitution: from (3)'', C = 1 . Substituting for C into (2)'$ gives B = 1 ,
and substituting for C into (1) gives A = 1 . So A = B = C = 1 , and
2 x 2 − x − 1 ( x 2 + 1 ) ( x − 2 ) = x + 1 x 2 + 1 + 1 x − 2 .
Thus
∫ 2 x 2 − x − 1 ( x 2 + 1 ) ( x − 2 ) d x = ∫ x + 1 x 2 + 1 d x + ∫ d x x − 2 .
Split up the first integral further:
∫ x x 2 + 1 d x + ∫ 1 x 2 + 1 d x + ∫ d x x − 2 .
In the first integral let w = x 2 + 1 , so d w = 2 x d x and d w 2 = x d x .
We get
1 2 ∫ d w w + ∫ d x x 2 + 1 + ∫ d x x − 2 = 1 2 ln ⁡ | w | + arctan ⁡ ( x ) + ln ⁡ | x − 2 | + C .
Converting back to x ,
1 2 ln ⁡ | x 2 + 1 | + arctan ⁡ ( x ) + ln ⁡ | x − 2 | + C .

Example

∫ x 3 − 7 x 2 + 10 x + 1 x 2 − 7 x + 10 d x .
Here the degree of the numerator is ≥ that of the denominator. As in the last
section, the first thing to do here is algebraic long division:
x 3 − 7 x 2 + 10 x + 1 x 2 − 7 x + 10 = x ( x 2 − 7 x + 10 ) + 1 x 2 − 7 x + 10 = x + 1 x 2 − 7 x + 10 .
If we factor x 2 − 7 x + 10 as ( x − 2 ) ( x − 5 ) , we get
x 3 − 7 x 2 + 10 x + 1 x 2 − 7 x + 10 = x + 1 ( x − 2 ) ( x − 5 ) .
From earlier,
1 ( x − 2 ) ( x − 5 ) = − 1 3 ( x − 2 ) + 1 3 ( x − 5 ) .
and so
x 3 − 7 x 2 + 10 x + 1 x 2 − 7 x + 10 = x − 1 3 ( x − 2 ) + 1 3 ( x − 5 ) .
Thus
∫ x 3 − 7 x 2 + 10 x + 1 x 2 − 7 x + 10 d x = ∫ x d x − 1 3 ∫ d x x − 2 + 1 3 ∫ d x x − 5
= x 2 2 − 1 3 ln ⁡ | x − 2 | + 1 3 ln ⁡ | x − 5 | + C .
Partial fractions works for many, but not all, rational functions.

Strategy for integrating a rational function P ( x ) Q ( x )

  1. If deg ⁡ P ≥ deg ⁡ Q , use algebraic long division and use partial fractions
    on the remainder.
  2. If Q ( x ) is a product of distinct linear factors, use partial fractions of
    the form A x − c .
  3. If Q ( x ) has a repeated linear factor ( x − c ) n , use partial fractions of
    the form
    A 1 x − c + A 2 ( x − c ) 2 + ⋯ + A n ( x − c ) n .
  4. If Q ( x ) contains an unfactorable quadratic factor q ( x ) , try a partial
    fraction of the form A x + B q ( x ) .
We will see more about how to integrate factors of the type A x + B q ( x )
soon.
