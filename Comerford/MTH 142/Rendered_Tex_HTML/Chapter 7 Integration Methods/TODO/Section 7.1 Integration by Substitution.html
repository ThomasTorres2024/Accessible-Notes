§ 7.1   Integration by Substitution

Consider the following indefinite integral:

$$\int 3x^2\cos\left(x^3\right)\,dx.$$

The integrand \(3x^2\cos(x^3)\) looks like the derivative of a composite function, which comes from the chain rule.

Indeed,

$$\frac{d}{dx}\bigl(\sin(x^3)\bigr) = \cos(x^3)\,\frac{d}{dx}(x^3) = \cos(x^3)\,3x^2.$$

\(\text{Reminder:}\; \dfrac{d}{dx}\bigl(f(g(x))\bigr) = f'\bigl(g(x)\bigr)\,g'(x)\)

So \(\sin(x^3)\) is an antiderivative of \(3x^2\cos(x^3)\), and thus

$$\int 3x^2\cos\left(x^3\right)\,dx = \sin\left(x^3\right) + C.$$

More generally, if \(f\) and \(g\) are functions and \(f\) has an antiderivative \(F\), then by the chain rule,

$$\frac{d}{dx}\bigl(F(g(x))\bigr) = F'\bigl(g(x)\bigr)\,g'(x) = f\bigl(g(x)\bigr)\,g'(x), \qquad \text{since } F' = f.$$

Thus \(F(g(x))\) is an antiderivative of \(f(g(x))\,g'(x)\), and so

$$\int f\bigl(g(x)\bigr)\,g'(x)\,dx = F\bigl(g(x)\bigr) + C.$$

Strategy is to spot the chain rule pattern in the integrand—i.e., find \(f\) and \(g\). Best bet is usually to go for the inside function \(g\) first.

For example, in the last example, \(g(x)=x^3\).



Examples

  1. $$\int 2e^{2x}\,dx = e^{2x}+C,\qquad g(x)=2x.$$
  2. $$\int 2x\cos\bigl(x^2+1\bigr)\,dx = \sin\bigl(x^2+1\bigr)+C,\qquad g(x)=x^2+1.$$
  3. $$\int 2t\,e^{t^2+1}\,dt = e^{t^2+1}+C.$$


When the pattern isn’t perfect

Consider

$$\int x^2\cos\bigl(x^3\bigr)\,dx.$$

Recall from before that

$$\frac{d}{dx}\bigl(\sin(x^3)\bigr)=3x^2\cos\bigl(x^3\bigr).$$

We are out by a factor of \(3\), and we can get round this by writing

$$\int x^2\cos\bigl(x^3\bigr)\,dx = \frac{1}{3}\int 3x^2\cos\bigl(x^3\bigr)\,dx = \frac{1}{3}\sin\bigl(x^3\bigr)+C.$$
